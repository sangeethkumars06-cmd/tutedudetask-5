#  Task 5 - Laundry Services Hero Section

## Overview
This project is a **Laundry Services Hero Section** built using **HTML5** and **CSS3**. It features a clean navigation bar and an attractive hero section designed without using Flexbox, following the assignment requirements.

---

##  Features
- Responsive navigation bar
- Logo displayed on the left
- Navigation links:
  - Home
  - Services
  - About Us
  - Contact Us
- Username section on the right
- Hero section with:
  - Eye-catching heading
  - Brief service description
  - "Book a Service Today!" button
  - Laundry-themed image
- Clean and modern UI design

---

## Technologies Used
- HTML5
- CSS3

---

## CSS Concepts Applied
- Display (`inline-block`)
- Float
- Position
- Viewport Units (`vh`)
- Box Model
- Hover Effects
- Text Styling
